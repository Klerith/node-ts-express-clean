# Dev

1. Clonar archivo ```.env.template``` por ```.env```
2. Pueden usar esas variables de entorno
3. Levantar la base de datos
```
docker compose up -d
```
4. Instalar las dependencias ```npm install```
5. Correr en modo de desarrollo ```npm run dev```
